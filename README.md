# Production Auth System

A full-stack authentication system built with React, Vite, and Express. This project demonstrates user registration and login flow with a clean UI and modular backend structure.

---

## Features

* User Signup & Login UI
* Modern React + Vite frontend
* Express-based backend
* Environment-based configuration
* Clean and scalable project structure

---

## Tech Stack

* Frontend: React, Vite
* Backend: Node.js, Express
* Styling: Tailwind CSS
* Other: dotenv

---

## 📦 Installation

Clone the repository:

git clone https://github.com/your-username/your-repo.git

Go into the project folder:

cd your-repo

Install dependencies:

npm install

---

##  Run the Project

Start development server:

npm run dev

Then open:

http://localhost:3000

---

##  Notes

* Authentication backend/API may require additional configuration
* Environment variables should be set in a `.env` file
* `.env` is not included for security reasons

---

##  Project Structure

* `/src` → Frontend code
* `/server` → Backend (if present)
* `vite.config.js` → Vite configuration
* `.env` → Environment variables
